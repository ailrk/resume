# My resume


### How to build
All dependencies are packaged in `flake.nix`. If you use nix, do the following:
```
$ nix develop
$ make
```

If you don't use nix, you need to have `xelatex`, `pandoc` and the following latex packages:
```
xltxtra
xunicode
geometry
marginnote
multirow
sectsty
ulem
hyperref
polyglossia
fontspec
greek-fontenc
```
